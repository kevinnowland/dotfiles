# Dotfiles


## Installation

These are some of my dotfiles.

To use, clone to `~/.dotfiles` and create the following symlinks:

```bash
ln -s ~/.dotfiles/zsh/zshrc ~/.zshrc
ln -s ~/.dotfiles/vim/vimrc ~/.vimrc
ln -s ~/.dotfiles/tmux/tmux.conf ~/.tmux.conf
```

## Things to install

1. tmux
2. pyenv
3. tfenv
4. docker
